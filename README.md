# Wazuh_OPNsense Grafana Dashboard

This repository contains a Grafana dashboard for visualizing and monitoring security events from **Wazuh** and **OPNsense**. The dashboard integrates data from both systems to provide a comprehensive view of network security, firewall logs, and system events in real-time.

## Features
- **Wazuh Integration**: Visualize security event logs, alerts, and threat intelligence data collected by Wazuh.
- **OPNsense Integration**: Monitor firewall logs, system metrics, and VPN connections from OPNsense.
- **Real-Time Monitoring**: Dashboards for real-time tracking of security events and system health.
- **Customizable Panels**: Tailor the dashboard to suit specific monitoring needs with customizable Grafana panels.
